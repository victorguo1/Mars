Cloud Computing is coming, rapidly and strong. It’s in banking, government, start-ups, small business and everywhere. Azure is one of the leading cloud computing providers.

40 hours, lectures and lots of on hands on labs.
Objectives
  • Computing: Virutal Machine,Application Load Balance,Auto scaling,Azure Function, WebJob
  • Storage: Blob, File, Queue, Table
  • Databases: SQL database, Redis Caching
  • Networking: Virutal Network, Azure CDN, Traffic Manager, Network Security
  • management Tools: Azure Template, Azure CLI, Azure Monitoring
  • Application Integration: Messaging Queue,Event Hub,Service Bus, Notificaiton Service
  • Security & Identify: Azure Active Directory, SSO
What are the requirements?
  • You will need to set up an free Azure Account (you can use the free tier for this course)
  • Your own domain name (optional, but recommended)
  • A Windows, Linux or Mac PC/Laptop
What is the target audience?
  • Absolute Beginners. No prior cloud experience necessary
  • University students interested in get into Cloud computing field
  • Programmers Interested in Deploying Applications on Azure
  • System Admin and DBA interested in expanding skills to Azure
  • Existing Solutions Architects
