Cloud Computing is coming, rapidly and strong. It’s in banking, government, start-ups, small business and everywhere. AWS is one of the leading cloud computing providers.

40 hours, lectures and lots of on hands on labs.
Objectives
  • Gain hands-on experience to create a solution to build using architecture best practices.
  • Gain hands-on with AWS Core Services: EC2, S3,ELB,EFS, RDS, NoSQL, S3,IAM, VPC.
  • Gain knowledge of application architecture, networking, security, reliability and scalability concepts.
  • Gain understanding of the AWS global infrastructure.
What are the requirements?
  • You will need to set up an free AWS Account (you can use the free tier for this course)
  • Your own domain name (optional, but recommended)
  • A Windows, Linux or Mac PC/Laptop
What is the target audience?
  • AWS Absolute Beginners. No prior AWS experience necessary
  • University students interested in get into Cloud computing field
  • Programmers Interested in Deploying Applications on AWS
  • System Admin and DBA interested in expanding skills to AWS
  • Existing Solutions Architects
